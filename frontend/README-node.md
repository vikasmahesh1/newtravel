Node version recommendation

This project requires Node >= 22.12.0 for the frontend (Vite 7). Use nvm or nvm-windows to install and switch versions.

Windows (nvm-windows):
1. Install nvm-windows from https://github.com/coreybutler/nvm-windows/releases
2. Run:

   nvm install 22.12.0
   nvm use 22.12.0

macOS / Linux (nvm):
1. Install nvm: https://github.com/nvm-sh/nvm
2. Run:

   nvm install 22.12.0
   nvm use 22.12.0

Also see the `.nvmrc` file at the project root of the frontend for the recommended version.
